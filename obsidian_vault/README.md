# 쿠팡 도서 판매 자동화 시스템 개발 노트

## 📚 Obsidian Vault 구조

```
obsidian_vault/
├── 00-Index/              # 목차 및 대시보드
├── 01-Daily/              # 일일 개발 로그
├── 02-Features/           # 기능별 문서
├── 03-Technical/          # 기술 문서
├── 04-Decisions/          # 의사결정 로그
├── 05-Tasks/              # 작업 추적
└── 06-Database/           # DB 스키마
```

## 🔗 주요 페이지

- [[Index]] - 전체 시스템 대시보드
- [[Development Timeline]] - 개발 타임라인
- [[Tech Stack]] - 기술 스택
- [[Database Schema V2]] - DB 스키마

## 🏷️ 태그 체계

- `#feature` - 기능 구현
- `#bug` - 버그 수정
- `#refactor` - 리팩토링
- `#test` - 테스트
- `#docs` - 문서화
- `#database` - 데이터베이스
- `#api` - API 연동

## 📝 작성 규칙

1. 모든 파일은 마크다운 (.md)
2. 백링크 활용 [[페이지명]]
3. 태그는 소문자 #tag
4. 일일 로그는 YYYY-MM-DD 형식
