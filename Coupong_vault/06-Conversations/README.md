## 📁 폴더 용도

Claude Code와 나눈 **중요한 대화**를 보관하는 폴더입니다.

### 기록하는 내용
- 🎯 프로젝트 관련 중요 논의
- 💡 기술적 의사결정 과정
- 🐛 복잡한 문제 해결 과정
- 📚 학습 및 인사이트
- 🔧 시스템 설계 논의

### 기록하지 않는 내용
- 일상적인 짧은 질문
- 단순 코드 수정
- 간단한 버그 수정

일일 개발 로그는 [[01-Daily/]]에 기록됩니다.

## 💡 활용 방법

### 1. 문제 해결 시 참조
```
비슷한 문제를 예전에 해결한 적이 있나?
→ Conversations 폴더 검색
→ 이전 해결 방법 참고
```

### 2. 의사결정 근거 확인
```
왜 이렇게 설계했지?
→ 관련 대화 찾기
→ 당시 논의 내용 확인
```

### 3. 학습 자료
```
복잡한 개념을 어떻게 이해했지?
→ 대화 내용 다시 읽기
→ 단계별 이해 과정 복습
```

### 4. 온보딩 자료
```
새로운 팀원에게:
"이 대화들을 읽어보세요"
→ 프로젝트 맥락 빠르게 이해
```
