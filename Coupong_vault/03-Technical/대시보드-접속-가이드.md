## 대시보드 사용법

### 메뉴 (왼쪽 사이드바)

| 메뉴 | 설명 |
|------|------|
| **등록 현황** | 계정별 쿠팡 등록 상품 확인, 검색, 수정 |
| **신규 등록** | 새 상품 쿠팡에 등록 (알라딘 크롤링 → 등록) |
| **전체 현황** | 전체 계정/출판사/마진 요약 |
| **매출 분석** | 매출 추이, 베스트셀러, 계정별 비교 |

### 매출 분석 페이지

1. **기간 선택**: 1주 / 1개월 / 3개월
2. **계정 선택**: 전체 또는 특정 계정
3. **매출 동기화** 버튼: 쿠팡에서 최신 매출 데이터 가져오기
4. 화면 구성:
   - **핵심 지표 3개**: 총 매출, 정산금액, 주문 수 (전기 대비 증감 %)
   - **인사이트**: 매출 증감, 베스트셀러, 최고 매출 계정 자동 요약
   - **일별 매출 그래프**
   - **베스트셀러 탭**: 잘 팔리는 상품 순위 + 광고 추천
   - **계정 비교 탭**: 계정별 매출 비교 (전체 선택 시)
   - **상세 분석 탭**: 상품별 판매/환불 현황 (계정 선택 시)

### 신규 등록 페이지

1. **크롤링 시작** → 알라딘에서 새 책 검색
2. 목록에서 등록할 상품 **체크**
3. **Dry Run** 체크 → 테스트만 (실제 등록 안 함)
4. **Dry Run** 해제 → 실제 쿠팡 등록

## 문제 해결

### "사이트에 연결할 수 없음"

| 원인 | 해결 |
|------|------|
| Tailscale 꺼져 있음 | 폰/PC에서 Tailscale 앱 열고 연결 확인 |
| 서버 PC 꺼져 있음 | 서버 PC 켜기 (자동으로 대시보드 시작됨) |
| 서버 PC 절전모드 | 마우스/키보드 눌러서 깨우기 |

### "DB 오류" 표시

- 보통 새로고침하면 해결됨
- 매출 분석 → "매출 동기화" 버튼 한 번 누르기

### 대시보드가 느림

- 3개월 데이터 조회 시 약간 느릴 수 있음
- 1주 / 1개월로 기간 줄이기
