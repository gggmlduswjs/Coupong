## 가격 수정 원인

알라딘 API 크롤링 시 priceSales(판매가, 이미 10% 할인됨)를 list_price(정가)로 저장하는 오류가 있었습니다.

**예시:**
- 정가 19,000원 도서

- 이중 할인: 17,100 x 0.9 = 15,390원 (판매가)
- 정상 판매가: 19,000 x 0.9 = 17,100원
- 손실: 1,710원/권 (약 10%)
