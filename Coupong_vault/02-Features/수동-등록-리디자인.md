# 수동 등록 페이지 리디자인

#feature #dashboard #registration

**상태:** ✅ 완료
**구현 파일:** `dashboard.py` (수동 등록 섹션)

---

## 개요
수동 등록 페이지를 쿠팡 셀러센터 스타일의 5-카드 섹션 레이아웃으로 리디자인. 번호 매긴 단계별 흐름, 필수 필드 표시, 마진 미리보기, 자동생성 필드 프리뷰 등 추가.

## 5-카드 섹션 구조

| # | 섹션 | 주요 요소 |
|---|------|----------|
| 1 | 카테고리 선택 | 코드 입력 + 추천 API + 카테고리명 표시 |
| 2 | 기본 정보 | ISBN 조회(DB/알라딘) + 상품명/저자/출판사 입력 + help 텍스트 |
| 3 | 판매 정보 | 정가/판매가/과세유형/배송비 4열 + 마진 metric 미리보기 |
| 4 | 이미지/상세 | 이미지 미리보기 + 설명 2열 + 고시정보/속성/검색태그 프리뷰 |
| 5 | 등록 계정 + 검토 | data_editor 테이블 + 검증 요약 + 페이로드 미리보기 + 등록 버튼 |

## 신규 추가 요소

### 마진 미리보기 (섹션 3)
- 쿠팡 수수료 (11%), 배송 원가 (₩2,300), 예상 순마진, 할인율
- 적자 시 경고 메시지 표시

### 자동생성 필드 미리보기 (섹션 4)
- `_build_book_notices()` → 상품고시정보 7개 항목
- `_build_book_attributes()` → 필수 속성 5개 항목
- `_generate_search_tags()` → 검색 태그 최대 20개 (pill 뱃지)

### 계정 선택 테이블 (섹션 5)
- `st.data_editor` + CheckboxColumn으로 체크박스 테이블
- 선택 개수 실시간 표시

### 등록 전 검증 (섹션 5)
- 필수 항목 5개 (상품명/ISBN/정가/판매가/계정) 충족 여부 표시
- 미충족 시 warning, 충족 시 success

## CSS 스타일
- 섹션 헤더: 파란 하단 보더 + 원형 숫자 뱃지
- 검색 태그: pill 뱃지 (파란 배경)
- 검증 체크: 초록 체크/빨간 X
