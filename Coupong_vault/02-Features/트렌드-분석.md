# 트렌드 분석

#feature #dashboard #analytics

**상태:** ✅ 완료
**구현 파일:** `dashboard.py`

---

## 개요
대시보드 "트렌드" 페이지. 기존 "매출" 페이지가 기간별 현황 조회라면, "트렌드"는 추세·비교·예측에 집중하는 고급 분석 페이지.

## 구현 파일
- `dashboard.py` — `elif page == "트렌드":` 블록 (4개 탭)

## 핵심 로직

### 탭 A: 월별 추이·예측
- revenue_history에서 월별 매출/주문수/정산 집계
- pandas rolling(3)으로 3개월 이동평균(MA3)
- numpy polyfit(1차)으로 향후 3개월 예측선
- Plotly 복합 차트: 매출 bar + MA3 line + 예측 dashed line
- KPI: 최근3M vs 이전3M 성장률, 평균 월매출, 3개월 후 예측

### 탭 B: 출판사 성과
- 최근 3개월 vs 이전 3개월 출판사별 매출 비교
- 성장률 계산 → Plotly horizontal bar (top 10, 초록/빨강 색상)
- 매출 기여도 pie chart
- 테이블: 출판사, 최근매출, 이전매출, 성장률(%), 주문수

### 탭 C: 계정 성장
- 5개 계정 월매출 → Plotly multi-line chart
- 계정별 최근 3개월 성장률 bar chart
- 등록 상품수 vs 매출 scatter plot

### 탭 D: 상품 수명주기
- 상품별 첫판매일, 최근판매일, 총주문수, 활동월수 집계
- 4분류: 신규(30일 미만) / 성장(최근 증가) / 안정(꾸준) / 쇠퇴(최근 감소)
- KPI 카드 + pie chart + 필터 테이블

## 의존성
- `numpy` (polyfit — pandas 의존성으로 이미 설치됨)
- `plotly` (이미 사용 중)
- 추가 패키지 없음
